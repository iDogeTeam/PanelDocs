# error_code 说明

| Code     | Error              |
| -------- | ------------------ |
| 400      | 邮箱或密码错误       |
| 401      | 权限不足            |
| 404	   | 不存在的资源/接口  |
| 414	   | 需要用户介入提供验证/资料补全 |
| 421	   | 用户操作失败（不符合规范的数据，可能是故意制造绕开前台限制）	|
| 413	   | 用户操作失败（不通过的合理请求，例如已经签到）|
| 500      | 服务器内部错误       |
